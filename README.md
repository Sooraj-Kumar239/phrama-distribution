# Pharma Distribution Management System

## Project Overview
This project is a full-stack web application developed to manage pharmaceutical distribution operations efficiently.

The main goal of this system is to allow businesses to purchase products from distributors and sell them to customers while managing employees, designations, and overall operations through a centralized dashboard.

---

## Tools and Technologies Used
- React.js – Frontend user interface
- Node.js (Express) – Backend API
- MySQL (Railway) – Database
- Azure – Application deployment
- GitHub – Version control

---

## Project Architecture
The workflow of the project is structured as follows:

- User interacts with the React frontend  
- Frontend sends requests to Node.js backend APIs  
- Backend processes requests and communicates with MySQL database  
- Data is stored/retrieved from the database  
- Response is sent back to frontend and displayed to user  

This structure ensures smooth and efficient system operation.

---

## Features
- Product Management (Add, Update, Delete)
- Customer Management
- Purchase System (from distributors)
- Sales System (to customers)
- Employee & Designation Management
